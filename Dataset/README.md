# Dataset

## Source

Healthcare Analytics Patient Flow Dataset (Kaggle)

Original Dataset:
https://www.kaggle.com/datasets/hassanjameelahmed/healthcare-analytics-patient-flow-data

## Description

This project uses a cleaned version of the original dataset for SQL analysis, Python analysis.

The original dataset contains:

- 9,216 patient visits
- Period: September 2023 – December 2024

The cleaned dataset was generated during the data preparation process and includes standardized column names and data types.
